Marks a method as a Command, automatically registering it with the command system and making it visible in the plugin installer. All registered plugin commands can also be seen by typing /lunaris help.

[AttributeUsage(AttributeTargets.Method)]
public sealed class LunarisCommandAttribute : Attribute, _Attribute

Examples

[LunarisCommand("heal", "Heals the player by the specified amount")]
public void HealCommand(float amount)
{
    player.Heal(amount);
}

// In chat: /myplugin heal 50

Remarks

The command is registered in chat as /pluginname commandname, where pluginname comes from the LunarisPluginAttribute name (spaces removed and converted to lowercase). The method's parameters determine the command's arguments, and only the following types are supported:

  • string
  • int
  • float
Warning

The method must match a supported parameter signature (0-8 args) and params.

  • Using unsupported types will throw an ArgumentException.
  • Using more than 8 parameters will throw a NotSupportedException.
